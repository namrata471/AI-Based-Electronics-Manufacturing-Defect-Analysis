# AI-Based Electronics Manufacturing Defect Analysis System

## Project Overview

This project predicts the severity of defects in electronics manufacturing using Machine Learning. It analyzes manufacturing parameters such as temperature, humidity, voltage, current, and AI risk score to classify defect severity.

The project also includes a Flask web application where users can enter manufacturing parameters and receive a real-time prediction.

A Machine Learning and Flask-based web application for predicting defect severity in electronics manufacturing.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Flask
- HTML
- CSS
- Joblib
- Visual Studio Code

## Features

- Performs data preprocessing and cleaning.
- Visualizes manufacturing and defect data.
- Trains a Random Forest machine learning model.
- Predicts defect severity (High, Medium, Low).
- Interactive web interface built using Flask.
- Real-time prediction based on user inputs.

## How to Run

1. Clone the repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Run the Flask application:

```bash
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000
```
